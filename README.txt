Universal Cookbook

A Chrome Extension. Simple online conversion between U.S. cups, imperial and metric.

How to Install

* Clone the repository
* From Chrome go to the Extension page (you can go through the menu or put chrome://settings/extensions in the address bar)
* Make sure you have "Developer mode" switched on
* Click "Load unpacked extension..." and select the repository directory
* You should see the Universal Cookbook in your list of extensions, make sure it is enabled
* A little weighing scale icon should appear to the right of your address bar

How to Use

When you're looking at a U.S. recipe that includes cup measurements that you'd like to translate...

* Click the little weigh scale icon to the right of your address bar
* All cup measurements will be automatically converted and highlighted. You can see their original values by hovering over them.
